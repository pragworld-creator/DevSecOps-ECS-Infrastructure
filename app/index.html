Featured Architecture: Enterprise DevSecOps Portfolio

This portfolio is not just a static webpage; it is hosted on a highly available, fault-tolerant, and securely isolated AWS architecture provisioned entirely via Terraform (IaC).

Infrastructure & Technology Stack:

  • Network Isolation: Custom VPC containing 4 subnets across 2 Availability Zones (AZs) for high availability.
  • The DMZ (Public Subnets): Hosts an Application Load Balancer (ALB) acting as the secure ingress point, guarded by strict Security Groups allowing only HTTP/HTTPS traffic.
  • The Vault (Private Subnets): Houses the compute layer. Completely isolated from direct internet access.
  • Compute & Orchestration: Containerized via Docker, stored in a private Amazon ECR registry, and orchestrated using Amazon ECS running on serverless AWS Fargate.
  • Security & IAM: Enforces the Principle of Least Privilege using decoupled Task Execution and Task Roles. Utilizes Security Group Chaining to ensure containers only accept traffic from the ALB.
  • PrivateLink: Implements VPC Endpoints to pull ECR images securely over the AWS backbone network without traversing the public internet.
